# 使用说明

## 安装node

项目开发环境 node 版本为10.16

## clone项目到本地，推荐使用ssh方式
```
git clone git@github.com:JackDing1208/vue-dropbox.git
```

## 进入工程目录安装依赖，推荐使用yarn
```
cd vue-dropbox
yarn install
```

### 本地开发环境跑项目
```
yarn start
```

### 打包到dist目录
```
yarn build
```
